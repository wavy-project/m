# m
📴

## hardware
- Rectangular 4K display with rounded corners
- Fingerprint reader on back side near camera
- Front camera
- Back camera
- 3.5mm headphone jack
- USB type-C port
- Call speaker
- Playback speaker
- Microphone
- Volume button
- Power button
